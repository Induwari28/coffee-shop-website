# ☕ Coffee Shop Website

A responsive Coffee Shop website built using HTML, CSS, and JavaScript.  
This project is designed to simulate a real-world café landing page with a clean UI and mobile-friendly design.

---

## 🚀 Features

- Fully responsive design (mobile, tablet, desktop)
- Interactive navigation menu
- Search bar toggle functionality
- Product showcase section with images and pricing
- Customer testimonials section
- Smooth animations and hover effects
- Modern UI layout inspired by real coffee shop websites

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid + Media Queries)
- JavaScript (DOM Manipulation)
- Boxicons (Icons library)

---

## 📱 Responsive Design

Works smoothly across:
- Mobile devices
- Tablets
- Laptops
- Desktop screens

---

## 🎯 What I learned

- Building responsive layouts from scratch
- Handling UI bugs (icons, layout overlap, responsiveness)
- DOM manipulation in JavaScript
- Structuring a real-world frontend project
